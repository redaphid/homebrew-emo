# Redaphid Emo

## How do I install these formulae?

`brew install redaphid/emo/<formula>`

Or `brew tap redaphid/emo` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "redaphid/emo"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
